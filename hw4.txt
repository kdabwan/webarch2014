(#2)………………………………
Explain what happens when your browser loads http://people.ischool.berkeley.edu/~<USER>/server/wiki from sending the request to loading HTML?

1. The browser sends DNS request to finding IP address of http://people.ischool.berkeley.edu/~<USER>/server
2. sends a connection request to that IP address and port 80 like the we did in telnet
3. sends  GET /wiki HTTP/1.1
4. receives a 302 return code and redirected to en.wikipedia.org
5. sends DNS request to finding IP address of en.wikipedia.org
6. sends a connection request to wikipedia IP address and port 80
7. sends  GET / HTTP/1.1
8. receives response from wikipedia depends whether wikipedia server is running or not
